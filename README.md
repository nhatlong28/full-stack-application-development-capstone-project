# Best Cars - Car Dealership Review Platform

Welcome to **Best Cars**, a comprehensive national car dealership review platform. This application allows users to browse dealerships across the country, view detailed reviews, and share their own experiences.

## Project Overview
This project is built as part of the IBM Full Stack Software Developer Professional Certificate. It integrates a Django backend with a React frontend and utilizes microservices for dealership management and sentiment analysis.

## Features
- **User Authentication**: Sign up and login to post reviews.
- **Dealership Directory**: Filter dealerships by state.
- **Sentiment Analysis**: Automatic sentiment analysis of user reviews.
- **CI/CD**: Fully integrated with GitHub Actions for continuous integration.

## Project Structure
- `server/`: Main application directory containing Django and React code.
- `server/djangoapp/`: Django application logic.
- `server/frontend/`: React frontend.
- `database/`: Microservices for dealership data.
